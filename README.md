# inkysite

Personal website for Inky. Built with plain HTML/CSS/JS, deployed on Cloudflare Pages.

## pages
- **hi** — landing page
- **experiences** — work history and skills
- **projects** — side projects
- **menu** — party menu

## stack
- static HTML/CSS/JS
- cloudflare pages (hosting)
- cloudflare functions (serverless, if needed)

## design notes
- **dual nav situation (todo)**: there are effectively two nav bars — the `#nav` bar shared across all pages (projects · experiences · menu), and the site-links list on the index.html main column that duplicates the same links. these should eventually be reconciled so there's one canonical navigation.
